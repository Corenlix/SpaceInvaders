# SpaceInvaders
Реализовано:
- Волны врагов на уровне
- Два типа противников
- Три уровня
- Три вида вооружения
- Накопление очков за убийства противников
- Рестарт уровня при смерти игрока и переход на следующий при убийстве всех врагов
